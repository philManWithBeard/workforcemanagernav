​Introduction

Welcome to the workforce manager. It's the new place that we'll be asking you to:

  • Record changes in FTE. You will only need to do this when there's a change rather than once a month.
  • Record Annual Leave, Sickness Leave and Other Leave.

​Recent Changes

  • 17/08/2022 - Added Long Term Sickness Leave Column to Leave Reporting pages
  • 17/08/2022 - We had "Add 'Phased Return' as a role" as an upcoming improvement, but decided not to implement it for architectural reasons. All leave-based adjustments should be recorded in the leave reporting page and Phased Return is a leave based adjustment, specifically a sickness leave adjustment
  • 10/08/2022 - Guider is removed from workforce search if you set their employment status to 'left employment'
  • 24/6/2022 - Added FTE column to LCA Workforce Search
  • 20/6/2022 - Removed the need to press the 'search' button on the LCA and DC Workforce Search pages. You only need to press it if you are looking to filter the results based on the criteria you select on the dropdowns.
  • 20/6/2022 - Write better guidance for Other Leave

​Upcoming Changes

  • The New Person form relies on OktaID, which is difficult to obtain. We will be removing this process entirely and replacing it with an automated integration with Okta e.g. Guider details will automatically appear.
  • Improving accuracy of the FTE report
  • ​Add description of roles and employment statuses
  • Leave reporting column titles not sticky
